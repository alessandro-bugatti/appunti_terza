# Determinazione del tipo di un triangolo

Scrivere una funzione che, dati come parametri 3 numeri che rappresentano i lati di un triangolo, **stampi a video** di che tipo di rettangolo si tratta, se _equilatero_, _isoscele_ o _scaleno_.

Successivamente scrivere un programma che chieda all'utente di inserire i tre numeri e stampi il tipo di triangolo. Il programma deve proseguire fino a quando desidera l'utente.
